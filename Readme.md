Вызов функции и добавление объекта в массив

Этот код определяет функцию addTask, которая:
Создает новую задачу с уникальным ID (используя crypto.randomUUID()), названием, описанием и временной меткой.
Добавляет эту задачу в глобальный массив tasks.
Демонстрирует добавление первой задачи и вывод всего списка задач в консоль.
