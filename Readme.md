1.# Управление списком задач

## Описание программы

Скрипт реализует логику добавления новых элементов в список дел через специализированную функцию. Основные этапы работы:

1.  **Создание задачи (`addTask`)**: Функция инкапсулирует процесс формирования объекта задачи. Она принимает базовые данные (название, описание) и дополняет их системными свойствами.
2.  **Генерация уникальных ID**: Для исключения конфликтов данных используется встроенный метод `crypto.randomUUID()`. Это гарантирует, что каждая задача получит неповторимый идентификатор.
3.  **Автоматизация метаданных**: Программа автоматически фиксирует точное время создания каждой записи (timestamp), что важно для сортировки и логирования.
4.  **Манипуляция массивом**: Скрипт демонстрирует наполнение глобального хранилища `tasks` путем добавления новых объектов в конец массива.

Результаты выполнения (первая созданная задача и итоговый список) выводятся в консоль для проверки корректности работы структуры данных.

## Как запустить

Для работы кода необходима среда, поддерживающая современные стандарты JavaScript и Web API:

- **В браузере**: Откройте консоль разработчика (F12) и вставьте код.

2.# Обработка данных о смартфонах (JavaScript)

Простой учебный проект на JavaScript, демонстрирующий базовые операции с массивами объектов: фильтрацию, маппинг и применение условной логики.

## Описание программы

Скрипт выполняет следующие действия:

1.  **Фильтрация**: Из исходного списка смартфонов выбираются только те, цена которых превышает 15 000 руб. (используется метод `.filter()`).
2.  **Обработка данных**: Создается новый массив на основе отфильтрованных данных (используется метод `.map()`). В процессе обработки к каждому объекту применяются:
    - **Динамическая скидка**: Размер скидки (5%, 10% или 15%) зависит от бренда телефона (iPhone, Samsung или другой).
    - **Категоризация**: Телефонам присваивается категория ("Premium", "Middle", "Budget") в зависимости от их финальной цены со скидкой.

Результаты выводятся в консоль в виде таблиц для удобного просмотра.

## Как запустить

Для запуска кода достаточно открыть файл в любой современной среде выполнения JavaScript (например, Node.js или прямо в браузере через тег `<script>`).
